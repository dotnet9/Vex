# 鳴謝

Vex 建立在 .NET 與開源桌面生態的工作成果之上。

感謝以下專案：

- Avalonia 提供跨平台 UI 框架。
- Prism 與 DryIoc 提供模組化應用程式組合能力。
- ReactiveUI 提供 ViewModel 開發模式。
- Semi.Avalonia 與 Ursa.Avalonia 提供桌面 UI 主題與視窗體驗。
- AvaloniaEdit 提供原始碼編輯器。
- Markdig 提供 Markdown 解析能力。
- SkiaSharp 與 Avalonia 渲染元件提供圖片與 PDF 匯出的基礎。
- CodeWF.EventBus、CodeWF.Markdown、CodeWF.AvaloniaControls 與其他 CodeWF 函式庫支援本專案。

Vex 由 CodeWF 出品。
