# 謝辞

Vex は .NET とオープンソースのデスクトップエコシステムの成果の上に構築されています。

以下のプロジェクトに感謝します。

- Avalonia: クロスプラットフォーム UI フレームワーク。
- Prism と DryIoc: モジュール化されたアプリケーション構成。
- ReactiveUI: ViewModel パターン。
- Semi.Avalonia と Ursa.Avalonia: デスクトップ UI テーマとウィンドウ体験。
- AvaloniaEdit: ソースエディター。
- Markdig: Markdown 解析。
- SkiaSharp と Avalonia のレンダリングコンポーネント: 画像および PDF エクスポートの基盤。
- CodeWF.EventBus、CodeWF.Markdown、CodeWF.AvaloniaControls、および本プロジェクトで使用しているその他の CodeWF ライブラリ。

Vex は CodeWF により提供されています。
